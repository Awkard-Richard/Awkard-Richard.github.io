HTML Tags

Element Description
<html> A self closing, optional tag that specifies to the browser the language used and how to structure the website.
<head> Contains information about a HTML page that isn't displayed on the page itself. This element contains metadata about the page including links to stylesheets and other resources such as google fonts etc.
<title> This element contains a text that definest the title of an HTML document. The title is displayed in the browser's title bar or tab in which the HTML page is displayed. This element can only be contained inside a document's <head> element.
<body> This element represents the content of the HTML document that displayed by the web browser.
<ol> Contains a list that is numbered
<ul> Contains a list that is not numbered i.e. dots
<li> This element contains the list data itself
<a> The almighty a (anchor) element is used to assign links to content on a HTML document.
<em> Content contained within em tags are italicized
<strong> This tag highlights important or serious content not to be overlooked.
<p> The p (paragraph) tags contain the bulk of the written content.
<img> This is the self-closing image tag within which images are embedded.
<video> This is the video tag within which images are embedded
<div> This element is used as a container that divides an HTML document into sections and is short for "division". <div> elements can contain flow content such as headings, paragraphs, links, images etc.
<br> The <br> element provides a line bread, or a blank line in text, particularly useful such as with postal addresses or in other circumstances where a gap in text is desirable.
<h1>-<h6> HTML can use six different levels of heading elements. The heading elements are ordered from the highest level <h1> to the lowest level <h6>.
<span> This element is an inline container for text and can be used to group text for styling purposes. However, as <span> is a generic container to seperate pieces of text from a larger body of text, its use should be avoided if a more semantic element is available.
<!-- --> In HTML, comments can be added between an opening <!-- and closing --> . content inside of comments will not be rendered by browsers, and are usually used to describe a part of code or provide other details. Comments can span single or multiple lines.
